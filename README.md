# AI Hallucination Detection System (VeriTru.AI)

A premium, full-stack AI safety application designed to detect, verifying, and score factual claims in LLM-generated content.

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- Python (v3.9+)

### 1. Backend Setup (Flask)
```bash
cd server
pip install -r requirements.txt
python app.py
```
*Server will start on http://localhost:5000*

### 2. Frontend Setup (React)
Open a new terminal:
```bash
cd client
npm install
npm run dev
```
*Client will start on http://localhost:5173*

## 🧠 Core Features
- **Atomic Claim Extraction**: Breaks text into verify-able facts.
- **Hallucination Scoring**: Weighted algorithms to detect high-risk content.
- **Premium UI**: Glassmorphism design with Framer Motion animations.
- **Demo Mode**: Works without an API key for impressive interviews (simulates AI verification).

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Chart.js
- **Backend**: Python, Flask, REST API
- **AI Logic**: Heuristic Analysis & Prompt Engineering
